Map Samples | USACE Blue Marsh Lake Recreation Guide



This application was developed to provide users the ability to explore recreational activities and areas that USACE Blue Marsh Lake Park has to offer both before they arrive and also while onsite. This site is mobile and desktop friendly. With lake attractions, comfort stations, access areas, pavilions, boat launches, and trails, there is plenty of information for visitors to become acquainted with the park. There is also the ability for park visitors to record any hazards or issues they encounter at the park such as excess trash or branches blocking trails. This application was developed with Bootstrap v5.1, HTML, CSS, and JavaScript as the frontend frameworks, the ArcGIS API for JavaScript v4.23 for the mapping framework, and ArcGIS Online as the server and backend.
